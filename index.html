01 — whoami

Born in 1992, based in Türkiye. I studied Electrical and Electronics Engineering, then learned programming in college through C: reading files, drawing shapes on screen, and building small tools around electronics, circuits, boards, robotics, and MATLAB.

I started professionally in medical image processing, building Qt desktop software for fMRI visualization and analysis. Then I moved into sensors, mobile apps, and motion-tracking algorithms for physiotherapy and sports, using Python and Java heavily.

Later I worked at retail scale, building optimization algorithms with mathematical modeling, big data ETL pipelines, C#, PySpark, and Airflow. Today I work with global enterprise customers across Big Data, AI stacks, data pipelines, and automation.

I am a long-time CLI person. After failing to adopt Vim three times, Neovim finally clicked. Aerospace + Ghostty + Neovim + tmux + pi is still the fastest developer experience for me in 2026. I try nearly every developer and AI tool I can find; keymap-driven workflows still feel lighter, faster, and more precise.
